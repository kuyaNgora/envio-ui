# Installation
Run one of the following commands to add `Envio UI` to your project:

<Source code={
`
# npm
npm install @envio-ui/core @envio-ui/icons @envio-ui/helpers @envio-ui/react styled-components

# yarn
yarn add @envio-ui/core @envio-ui/icons @envio-ui/helpers @envio-ui/react styled-components
`
} dark />

### Inter Font
`Envio UI` is designed to use the Inter font by default. You may add it to your project with npm via Fontsource, or with the Google Fonts CDN.
<Source code={
`
# npm
npm install @fontsource/inter

# yarn
yarn add @fontsource/inter
`
} dark />
