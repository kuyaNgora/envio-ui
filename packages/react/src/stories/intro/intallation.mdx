# Installation
Run one of the following commands to add `Envio UI` to your project:

<Source code={
`npm install @envio-ui/core @envio-ui/icons @envio-ui/helpers @envio-ui/react`
} dark />

<Source code={
`yarn add @envio-ui/core @envio-ui/icons @envio-ui/helpers @envio-ui/react`
} dark />
