# Input
An Input is a form field that a user can type a line of text into.

<b>Input</b> can have a placeholder, a default (initial) value, a label and a description.

~~~
### Usage
#### Simple
<Canvas>
    <Story of={InputStories.Simple} />
</Canvas>
<Source
    code={`
import React from "react";
import { Input, InputProps } from "@envio-ui/design";

export const Example: React.FC<InputProps> = () => {
    const [value, setValue] = React.useState<string>('');

    return (
        <Input
            label=""
            required
            value={value}
            onChange={(e) => setValue(e?.target?.value)}
        />
    );
};
    `}

    dark
    language="tsx"
/>
