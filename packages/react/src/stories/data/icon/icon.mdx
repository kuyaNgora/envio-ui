# Icons

Iconography uses symbols to represent meaning visually. They communicate a message and should be distinct and informative. Icons should be simple, yet clear enough to grab attention.

### Instalation (for standard modern project)
<Source code={`yarn add @envio-ui/icons`} language="bash" dark type="dynamic" />

### Usage
<Source code={`
import {IconAlertCircleOutline} from '@envio-ui/icons'

const Example = () => {
    return (
        <IconAlertCircleOutline size={30} fill="#000000">
    )
}`} type="code" dark language="tsx" />
